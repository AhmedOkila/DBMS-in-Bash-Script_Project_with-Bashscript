# **This is a DBMS Project made by bash Shell Script**

![alt]()

## **Usage :**
### 1. Clone the Project
```ruby
git clone git@github.com:AhmedOkila/DBMS_in_Bash_Script.git````
### 2. cd to it
```ruby
cd DBMS-Project/
```
### 3. Run the my_dbms script
```ruby
chmod +x *
./my_dbms.sh
```
